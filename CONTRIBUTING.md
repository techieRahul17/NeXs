# Contributing Guidelines

Thank you for considering contributing to this project!

## How to Contribute
- Fork the repository
- Create a feature branch (`git checkout -b feature-name`)
- Commit your changes
- Open a Pull Request

## Coding Standards
- Use clear naming conventions
- Follow project structure
- Add comments/documentation when needed

## Reporting Issues
Open an issue with:
- Description
- Steps to reproduce
- Expected behavior
- Screenshots (if needed)
